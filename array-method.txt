Array এর চারটা গুরুত্বপূর্ণ Method
1. map()
==========
== array এর এলিমেন্টের উপরে একটা এক্সান নিয়ে নতুন একটা array return করে।যেমন আমাদের কাছে একটা array আছে।

const arr = [10,12,14,16,18,20]

arr.map(element => element *2) করে আমাদের রিটার্ন করে।
